# Light Dict
英単語・成句を辞書サイトで検索してポップアップ表示します。
Look up a selected word in an English-Japanese dictionary webservice
and show it in a pop-up window.

Webページ中やFirefox内蔵のPDFビューア内で英単語・成句を選択し、特定のキーを押すと検索されます。デフォルトではShiftキーを押すとWeblioで検索されます。2つめの辞書とキーを設定することもできます。

選択状態でツールバーのボタンを押すことでも検索できます。他のアドオンとの相性などでキーが入力できないときなどに使用してください。

選択しない状態でツールバーのボタンを押すと検索パネルが表示され、任意の文字列で検索できます。Webページ以外の単語を検索したいときなどに使用してください。

検索した単語は検索履歴として保存され、回数順・時間順に並び替えて表示できます。簡易的な単語帳としても利用できます。設定画面のボタンから、もしくはツールバーのボタンを押し、その中にある検索履歴ボタンから開くことができます

辞書サイトは、 Weblio・goo辞書・コトバンク・Urban Dictionary から選択できます。
